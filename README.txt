FitPulse Pro - Fitness SaaS Platform
=======================================

Developed by: Jean Dupont & Marie Lambert
Project Type: BTS DAI Frontend Project
Tech Stack: HTML5, CSS3 (Tailwind v4), JavaScript (ES6)

Project Structure:
------------------
index.html      - Landing Page (Hero, Features, Pricing)
login.html      - User Login
signup.html     - New Account Application
dashboard.html  - Performance Hub (Interactive Dashboard)
about.html      - Team and Mission
css/            - Global Styles
js/             - Core Interactions
assets/         - Brand Images & Visuals

Key Features:
-------------
- Real-time Biometric Sync (Mocked)
- AI Performance Analytics
- High-Performance Dark Aesthetic
- Fully Responsive Design (Mobile/Desktop)
- Interactive Dashboard Components

How to Navigate:
----------------
Start at index.html and follow the "Join Now" or "Login" routes to see the dashboard.
Use the sidebar in the dashboard to access other areas.
